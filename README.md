# Exvalio – Smart Goods Exchange Platform

Exvalio is a web-based platform designed to facilitate fair and efficient exchange of goods with valuation.
The system integrates algorithm-based matching, AI-powered item classification, and inspector-based verification
to support secure and transparent item-for-item exchanges.

## Core Features

- AI-based item classification (CNN )
- Algorithm-based matching
- Inspector assignment and verification
- Real-time chat between users
